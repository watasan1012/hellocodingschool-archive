# hellocodingschool-archive

## Githubのリポジトリを作成する

## Git cloneをする

ローカル環境にリポジトリを複製するコマンド

```sh
% git clone リポジトリURL
```

## gitignore

.gitignoreファイルを作成することでgit対象からファイルを外す

```sh
% echo ".DS_Store" > .gitignore
```

## git add コマンド

ファイルの中身をインデックスに追加する



## git commit

git commit は、追加、変更したファイルをGitに登録するためのコマンドをします。
